# SahayakSetu Admin Panel && Doctor Panel

SahayakSetu Admin Panel is the administrative dashboard of the “SahayakSetu” Healthcare Management System.  
It allows administrators to manage doctors, appointments, healthcare workflows, and system operations through a secure and responsive web interface.

## Live Demo

Admin Panel Live Link:  
https://sahayaksetu-admin.vercel.app/

## Features

- Admin Authentication (JWT Based) && Doctor Authentication
- Doctor Management
- Appointment Monitoring
- Dashboard Analytics
- Queue Management
- Secure Protected Routes
- Responsive User Interface
- Cloud Integrated Healthcare Management

## Tech Stack

### Frontend
- React.js
- Tailwind CSS

- React Router DOM
- JavaScript (ES6+)

### Backend
- Node.js
- Express.js
- JWT Authentication

### Database
